# UVI Desktop Notifier
<img src="./res/icons8-sun-96.png"><br /><br />
공공데이터 포탈 기상청 생활기상지수 조회서비스 오픈 API를 이용한 현재 지역 실시간 자외선 지수 알리미

<br />

## < Features >

<b>- 자외선 지수 및 권고사항 출력</b> 

    현재 시간 및 현재 지역에 따른 오늘 (현재), 내일, 모레 자외선 지수 예측값 및 현재 자외선 지수에 따른 권고사항 출력

<b>- 지역 탐지 설정</b> 

    현재 기기 위치에 따라 자동으로 지역 탐지 혹은 수동으로 지역 목록에서 선택 가능

    ---
    ※ 대략 3,700여 개의 대한민국 내 지역만 지원
